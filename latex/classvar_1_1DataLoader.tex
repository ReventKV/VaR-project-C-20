\doxysubsection{Detailed Description}
Загрузчик исторических данных из файлов. 

Фасад для чтения файлов. Открывает файл, использует \doxylink{classvar_1_1CsvReader}{Csv\+Reader} для парсинга и формирует объект \doxylink{classvar_1_1TimeSeries}{Time\+Series}.
