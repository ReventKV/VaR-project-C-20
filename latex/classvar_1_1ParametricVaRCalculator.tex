\doxysubsection{Detailed Description}
Калькулятор параметрического VaR (Delta-\/\+Normal / Variance-\/\+Covariance). 

Предполагает нормальное распределение доходностей активов. Использует аналитическую формулу\+: $ VaR = Value \times z_{\alpha} \times \sigma_p \times \sqrt{T} $
