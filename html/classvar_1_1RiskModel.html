Detailed Description

Математическое ядро моделирования рисков.

Отвечает за обработку сырых временных рядов, их выравнивание и построение ковариационной матрицы.

std::vector< TimeSeries > var::RiskModel::alignTimeSeries ( const std::vector< TimeSeries > &  inputs)
static

Выравнивает временные ряды по единой шкале дат.

Статический метод.

  1. Собирает объединение (Union) всех дат из всех активов.
  2. Если у актива нет цены на дату T, используется цена T-1 (Forward Fill). Это критически важно для корректного расчета корреляций между активами с разных бирж.

inputsИсходные временные ряды.
Returns
Вектор выровненных рядов (все имеют одинаковую длину и набор дат).
